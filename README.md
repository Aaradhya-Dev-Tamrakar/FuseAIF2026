# FuseAIF2026

FuseAIF2026 is the working repository for the Fusemachines AI Fellowship 2026 assignments. It collects weekly deliverables across data wrangling, SQL, APIs, agentic text-to-SQL, and statistical machine learning projects.

## Repository Map

- `M1/WK1/` - data wrangling and SQL exploration assets for the first week of the fellowship.
- `M1/WK2/` - a FastAPI and PostgreSQL customer management API with modular routers and Docker support.
- `M1/WK3/` - a text-to-SQL agentic system with a pipeline, Streamlit UI, evaluation artifacts, and submission files.
- `M1/WK4/` - a churn and customer lifetime value modeling project built around linear models and notebook deliverables.
- `M2/WK5/` - tree-based classification and regression work, including a trained churn pipeline artifact.
- `M2/WK6/` - probabilistic modeling and Bayesian inference assignment materials.
- `M2/WK7/` - customer segmentation using RFM features and clustering methods.
- `M2/WK8/` - forecasting work for S&P 500 time series analysis and model comparison.
